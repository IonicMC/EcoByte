# ♻️ BoteByte (Exhibit Demo Build)

BoteByte is an interactive kiosk-style recycling exhibit that encourages proper bottle disposal through a simple reward-based experience.  
This repository contains the **public demo build** of the software used for exhibit showcasing.

> ⚠️ Note: This repo is a sanitized version and does **not** include sensitive machine logic such as hardware control, camera verification, token redemption backends, or production credentials.

---

## ✨ Features (Demo Build)

- Fullscreen kiosk-style interface (PyQt6)
- Clean exhibit UI layout (title, QR code, stats display)
- QR code generator with optional center logo
- Demo simulation buttons for bottle detection and token generation
- Idle timeout reset behavior
